# AI Joke Generator

テーマを入力すると、AIが面白いジョークを自動生成するアプリケーションです。Claude APIを使用して、様々なテーマに合わせたジョークを日本語で作成します。

## 特徴

- AIによるジョーク自動生成
- 任意のテーマでジョーク作成
- シンプルで使いやすいUI
- グラデーションデザイン
- Enterキーで即座に生成

## 技術スタック

- React (JSX)
- Claude AI API
- Lucide React Icons
- Tailwind CSS

## 使い方

1. [joke-generator.jsx](joke-generator.jsx)を適切なReact環境にコピー
2. テーマを入力（例：猫、プログラミング、料理など）
3. 「ジョーク生成」ボタンをクリックまたはEnterキーを押す
4. AIが生成したジョークが表示される

## 主な機能

### テーマ入力
- 自由なテーマでジョークを生成
- 入力例：動物、職業、日常、技術など

### ジョーク生成
- Claude AIによる高品質なジョーク
- 日本語で分かりやすく
- 短くて覚えやすい内容

### 操作性
- Enterキーで素早く生成
- ローディング表示
- エラーハンドリング

## API設定

Claude API キーが必要です。使用する際は、適切にAPI認証を設定してください。

```javascript
headers: {
  "Content-Type": "application/json",
  "x-api-key": "YOUR_API_KEY",
  "anthropic-version": "2023-06-01"
}
```

## テーマ例

- **動物**: 猫、犬、パンダ
- **職業**: プログラマー、医者、先生
- **食べ物**: ラーメン、寿司、カレー
- **趣味**: 読書、旅行、ゲーム
- **技術**: AI、スマホ、ロボット
