# Matrix Camera

カメラ映像をリアルタイムで「マトリックス」風の緑色の文字エフェクトに変換するアプリケーションです。HTML5のCanvasとWebカメラAPIを使用した視覚効果アプリです。

## 特徴

- リアルタイムカメラ映像処理
- マトリックス風の緑色文字エフェクト
- 降り注ぐ文字アニメーション
- モバイル対応
- フロント/バックカメラ切り替え
- スクリーンショット機能

## 技術スタック

- HTML5
- Canvas API
- WebRTC（カメラアクセス）
- JavaScript
- CSS3

## 使い方

1. [matrix-camera.html](matrix-camera.html)をブラウザで開く
2. カメラへのアクセスを許可
3. リアルタイムでマトリックスエフェクトが適用される
4. 必要に応じてカメラを切り替え

## 主な機能

### リアルタイムエフェクト
- カメラ映像を緑色の文字に変換
- 降り注ぐマトリックス風アニメーション
- 60FPSでスムーズな動作

### カメラ操作
- スタート/ストップ
- フロントカメラ/バックカメラ切り替え（モバイル）
- 自動明度調整

### ビジュアル
- ネオングリーンの文字
- 暗い背景でコントラスト強調
- Courier Newフォント使用

## 操作方法

### ボタン
- **カメラ開始**: カメラの起動
- **カメラ切替**: フロント/バックカメラの切り替え

### モバイル対応
- タッチ操作対応
- レスポンシブデザイン
- 縦横画面両対応

## ブラウザ要件

- カメラアクセス可能なブラウザ
- Canvas API対応
- WebRTC対応
- 推奨：Chrome、Firefox、Safari（最新版）

## プライバシー

- カメラ映像はローカルで処理
- サーバーへの送信なし
- 録画・保存機能なし（ブラウザ上でのみ表示）

## ユースケース

- パーティーやイベントでの演出
- SNS用の面白い動画撮影
- 映画「マトリックス」のファンアート
- カメラエフェクトの実験
