# Solar System 3D

太陽系の惑星を3Dで可視化し、公転運動をシミュレーションするアプリです。惑星の動きをインタラクティブに観察できます。

## 特徴

- 8つの惑星の3D表示
- リアルな公転速度比
- ズーム・回転機能
- 速度調整
- 惑星選択機能
- 太陽の表示
- 軌道の可視化

## 技術スタック

- React (TypeScript)
- Canvas API
- Lucide React Icons
- 3Dレンダリング（2D Canvas上）
- アニメーションループ

## 使い方

1. [solar_system_3d.tsx](solar-system-3d/solar_system_3d.tsx)を適切なReact環境にコピー
2. 自動的に惑星が公転を開始
3. 再生/一時停止ボタンで制御
4. スピードスライダーで速度調整
5. ズームボタンで拡大・縮小
6. 惑星をクリックで選択

## 操作方法

### 再生制御
- 再生/一時停止ボタン
- リセットボタンで初期位置に

### ズーム
- ズームインボタン
- ズームアウトボタン
- スクロールでズーム（実装可能）

### 速度調整
- スライダーで0.1倍～5倍速
- リアルタイムで反映

### 惑星選択
- 惑星をクリック
- 選択した惑星をハイライト

## 視覚効果

### 太陽
- グラデーション表示
- 発光エフェクト
- 中心に固定

### 軌道
- 半透明の円
- 惑星ごとに表示

### 惑星
- カラフルな色分け
- サイズの違いを表現
- 名前ラベル

### 土星の環
- 独自の視覚表現
- 惑星と一緒に回転
