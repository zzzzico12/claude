# Mood Weather Tracker

天気をモチーフにした気分トラッカーアプリです。その日の気分を天気で表現し、過去の記録を振り返ることができます。

## 特徴

- 天気アイコンで気分を記録（快晴、晴れ、曇り、小雨、大雨）
- カレンダー形式での記録表示
- AIによる気分分析機能
- データの永続化（ローカルストレージ）
- 美しいアイコンとアニメーション

## 技術スタック

- React (TypeScript)
- Lucide Icons（天気アイコン）
- ローカルストレージAPI
- Tailwind CSS

## 使い方

1. [mood-weather-tracker.tsx](mood-weather-tracker.tsx)をReact環境で実行
2. 今日の気分を天気アイコンから選択
3. 記録が自動保存される
4. カレンダーで過去の気分を確認
5. AI分析ボタンで気分の傾向を分析

## 天気の意味

- 快晴: 最高の気分
- 晴れ: 良い気分
- 曇り: 普通の気分
- 小雨: 少し憂鬱
- 大雨: とても憂鬱

## 主な機能

- 日々の気分記録
- 過去の記録閲覧
- AI気分分析
- データの永続化
