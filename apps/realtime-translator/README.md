# Realtime Translator

リアルタイムで多言語翻訳ができるWebアプリケーションです。テキストを入力すると即座に複数の言語に翻訳されます。

## 特徴

- リアルタイム翻訳機能
- 複数言語同時翻訳
- モダンなグラデーションUI
- スムーズなアニメーション
- レスポンシブデザイン
- 美しいスクロールバー

## 技術スタック

- React (HTML埋め込み版)
- Babel（JSXトランスパイル）
- Tailwind CSS
- Google Fonts（Noto Sans JP）
- 翻訳API連携

## 使い方

1. [realtime-translator.html](realtime-translator.html)をブラウザで開く
2. 入力フィールドにテキストを入力
3. リアルタイムで複数言語に翻訳される
4. 翻訳結果をコピーして利用

## 対応言語（例）

- 日本語
- 英語
- 中国語
- 韓国語
- スペイン語
- フランス語
- その他多数

## 主な機能

- リアルタイム翻訳
- 多言語同時表示
- コピー機能
- 履歴機能
