# Constellation Finder

指定した場所と時刻で見える星座を検索・表示するアプリケーションです。主要都市のデータを使って、その時期に観測できる星座を確認できます。

## 特徴

- 主要都市の星座観測情報
- 季節別星座表示
- 星座の星の配置と線を視覚化
- 日時指定機能
- 赤経・赤緯による計算
- 観測可能時間帯の表示

## 技術スタック

- React (JSX)
- Lucide React Icons
- 天文計算アルゴリズム
- Canvas描画

## 使い方

1. [constellation-finder.jsx](constellation-finder.jsx)を適切なReact環境にコピー
2. 都市を選択（東京、大阪、ニューヨーク、ロンドンなど10都市）
3. 日時を指定
4. 観測可能な星座が表示される
5. 星座をクリックで詳細表示

## 主な機能

### 都市選択
- 東京、大阪、札幌、福岡
- ニューヨーク、ロンドン、パリ
- シドニー、ソウル、北京

### 日時指定
- カレンダーで日付選択
- 時刻指定

### 星座表示
- 星の配置を視覚化
