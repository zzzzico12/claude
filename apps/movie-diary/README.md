# Movie Diary

観た映画を詳細に記録・管理できる日記アプリケーションです。感情スコア、視聴場所、一緒に観た人など、豊富な情報を記録できます。

## 特徴

- 映画の視聴記録を詳細に保存
- 感情スコアシステム（泣いた・笑った・ドキドキした）
- 年別統計・分析機能
- カラフルなポスター風カードUI
- データの永続化（ローカルストレージ）
- リスト表示と統計表示の切り替え

## 技術スタック

- React (TypeScript)
- Lucide React Icons
- ローカルストレージAPI
- Chart/グラフ表示機能

## 使い方

1. [movie-diary.tsx](movie-diary.tsx)を適切なReact環境にコピー
2. 「+」ボタンから新しい映画記録を追加
3. 詳細情報を入力して保存
4. リストで過去の記録を確認
5. 統計画面で年間の視聴傾向を分析

## 記録できる情報

### 基本情報
- **タイトル**: 映画のタイトル
- **視聴日**: 観た日付
- **ジャンル**: SF、恋愛、アクション、ホラー、コメディ、ドラマ、アニメ、ドキュメンタリー

### 視聴環境
- **場所**: 映画館、自宅、配信など
- **場所の種類**: theater, home, streaming
- **一緒に観た人**: 友達、家族、一人など

### 感情スコア（1-10段階）
- **泣いた**: 感動度
- **笑った**: 面白さ
- **ドキドキした**: 興奮度

### その他
- **気分タグ**: 感動、ワクワク、リラックス、興奮、癒し、学び
- **メモ**: 自由記述
- **ポスターカラー**: 8色から選択可能

## 主な機能

### リスト表示
- カラフルなカード形式で表示
- 感情スコアを視覚的に表示
- 年別フィルタリング
- 削除機能

### 統計表示
- ジャンル別視聴数
- 視聴場所の分布
- 気分タグの傾向
- 月別視聴数
- 平均感情スコア
- 総視聴数

### データ管理
- ローカルストレージでデータ永続化
- 自動保存機能
- データの読み込み・書き込み
