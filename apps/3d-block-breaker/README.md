# 3D Block Breaker

3Dグラフィックスを使用したブロック崩しゲームです。WebGLを活用し、美しいネオンエフェクトと立体的なゲームプレイを楽しめます。

## 特徴

- 3Dグラフィックスによる立体的なゲーム体験
- ネオンスタイルのビジュアルエフェクト
- スコアシステム搭載
- レスポンシブなパドル操作

## 技術スタック

- HTML5
- Canvas API / WebGL
- JavaScript
- Three.js（3Dレンダリング）

## 遊び方

1. [3d-block-breaker.html](3d-block-breaker.html)をブラウザで開く
2. マウスを左右に動かしてパドルを操作
3. ボールを跳ね返してブロックを破壊
4. すべてのブロックを破壊するとクリア

## ゲームの操作

- マウス移動: パドルを左右に移動
- クリック: ゲーム開始/リスタート

## スクリーンショット

ネオンカラーが輝く3D空間で、ブロック崩しの新しい体験をお楽しみください。
