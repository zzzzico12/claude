# Movie Pronunciation Practice

有名な映画のセリフを使って英語の発音練習ができるアプリケーションです。Web Speech APIを使用して、実際の発音を評価します。

## 特徴

- 有名映画のセリフで発音練習
- リアルタイム発音認識
- 精度スコア表示
- 日本語訳付き
- 音声再生機能
- カラフルな映画カードUI

## 技術スタック

- HTML5
- Web Speech API（音声認識）
- Speech Synthesis API（音声再生）
- JavaScript
- CSS3（グラデーション、アニメーション）

## 使い方

1. [movie-pronunciation-practice.html](movie-pronunciation-practice.html)をブラウザで開く
2. 映画のセリフを確認
3. 「🎤 音声で練習」ボタンをクリック
4. セリフを発音
5. 精度スコアを確認

## 主な機能

### 発音認識
- Web Speech APIによるリアルタイム認識
- 英語音声認識
- 精度スコア計算

### 音声再生
- ブラウザ標準の音声合成
- ネイティブ発音のお手本

### スコアリング
- 発音精度を％で表示
- 色分けフィードバック
  - 緑：高精度
  - 黄色：中程度
  - 赤：要練習

### UI/UX
- 映画ごとのカードデザイン
- グラデーション背景
- アニメーション効果
- レスポンシブデザイン

## ブラウザ要件

- Web Speech API対応ブラウザ
- 推奨：Chrome、Edge（最新版）
- マイク使用許可が必要

## 学習のコツ

1. **まず聞く**: 音声再生でお手本を確認
2. **繰り返す**: 何度も発音練習
3. **スコア確認**: 精度を確認して改善
4. **感情を込める**: 映画のシーンを想像しながら
