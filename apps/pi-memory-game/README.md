# Pi Memory Game

円周率（π）の小数点以下を暗記して入力するメモリーゲームです。どこまで正確に覚えられるかチャレンジしましょう！

## 特徴

- 円周率（小数点以下1000桁）の暗記ゲーム
- リアルタイム正誤判定
- タイマー機能
- 正解桁数カウント
- シンプルで集中しやすいUI
- モバイル対応

## 技術スタック

- React (TypeScript)
- Lucide React Icons
- Tailwind CSS

## 使い方

1. [pi-memory-game.tsx](pi-memory-game.tsx)を適切なReact環境にコピー
2. 「スタート」ボタンをクリック
3. 円周率の小数点以下を入力
4. 間違えるまで続ける
5. 結果を確認してリトライ

## ゲームルール

### 基本ルール
- 円周率は「3.」の後の数字から始まる
- 数字のみ入力可能
- 間違えた時点でゲーム終了
- 1000桁すべて正解すると完全クリア

### スコア
- 正解桁数が記録される
- 経過時間が表示される
- 間違えた数字も表示

## 円周率について

このゲームで使用している円周率：
```
3.1415926535897932384626433832795...
```
（小数点以下1000桁まで収録）

## 主な機能

### ゲーム状態管理
- ready: 開始前
- playing: プレイ中
- finished: 終了後

### リアルタイム判定
- 入力ごとに正誤を判定
- 即座にゲーム終了

### 結果表示
- 正解した桁数
- かかった時間
- 間違えた数字の表示

### UI/UX
- 自動フォーカス
- 大きな入力欄
- 見やすいフォント
- グラデーション背景

## 暗記のコツ

1. **区切って覚える**: 5桁ずつなど区切る
2. **語呂合わせ**: 数字を語呂合わせに変換
3. **繰り返し**: 何度も練習する
4. **視覚化**: イメージと結びつける
5. **段階的**: まず10桁、次に20桁...と増やす
