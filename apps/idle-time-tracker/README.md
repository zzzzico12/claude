# Idle Time Tracker

何もしていない時間（アイドル時間）を記録・可視化するアプリです。時間の使い方を見直し、生産性向上に役立てられます。

## 特徴

- アイドル時間の自動記録
- カレンダー形式での表示
- 日別・月別統計
- ヒートマップ可視化
- データの永続化
- 時間帯別分析

## 技術スタック

- HTML5
- JavaScript
- CSS3（グラデーション）
- ローカルストレージAPI
- カレンダーUI

## 使い方

1. [idle-time-tracker.html](idle-time-tracker.html)をブラウザで開く
2. アイドル時間を記録開始
3. カレンダーで日付を選択
4. その日のアイドル時間が表示される
5. 月間統計で傾向を確認

## 主な機能

### 時間記録
- 手動記録機能
- 時間帯の指定
- アイドル時間の分類

### カレンダー表示
- 月間カレンダー
- 日別のアイドル時間表示
- ヒートマップ色分け

### 統計機能
- 日別集計
- 週別集計
- 月別集計
- 平均時間計算

### データ管理
- ローカルストレージで保存
- データのエクスポート
- データの削除

## 可視化

### ヒートマップ
- 緑: アイドル時間少ない（生産的）
- 黄: 中程度
- 赤: アイドル時間多い（注意）

### グラフ
- 棒グラフで日別表示
- 折れ線グラフで推移表示
- 円グラフで時間帯別表示

## 記録できるアイドル時間

### カテゴリ例
- SNS閲覧
- 動画視聴
- ゲーム
- ぼーっとする
- スマホいじり
- その他

- 全データはローカル保存
- サーバーへの送信なし
- 個人のデバイス内で完結

## 改善のヒント

1. **パターン分析**: 特定の時間帯にアイドル時間が多い？
2. **目標設定**: 1日のアイドル時間目標を設定
3. **習慣改善**: 記録を見て無駄な時間を削減
4. **置き換え**: アイドル時間を有意義な活動に
