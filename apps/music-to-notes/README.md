# Music to Notes

マイクから音楽を入力して、リアルタイムで音符（ドレミファソラシド）に変換するアプリです。演奏した音を可視化できます。

## 特徴

- リアルタイム音声認識
- 音程検出（ピッチ検出）
- 音符への変換（C, D, E, F, G, A, B）
- 日本語表記（ド、レ、ミ、ファ、ソ、ラ、シ）
- 周波数表示
- 音符履歴の表示

## 技術スタック

- HTML5
- Web Audio API
- マイク入力（getUserMedia）
- ピッチ検出アルゴリズム
- JavaScript
- CSS3

## 使い方

1. [music_to_notes.html](music-to-notes/music_to_notes.html)をブラウザで開く
2. 「録音開始」ボタンをクリック
3. マイクへのアクセスを許可
4. 楽器を演奏または歌う
5. リアルタイムで音符が表示される
6. 「録音停止」で終了

## 検出される音符

### 基本音階（オクターブ4）
- C4 (ド): 約261.63 Hz
- D4 (レ): 約293.66 Hz
- E4 (ミ): 約329.63 Hz
- F4 (ファ): 約349.23 Hz
- G4 (ソ): 約392.00 Hz
- A4 (ラ): 約440.00 Hz
- B4 (シ): 約493.88 Hz

### 複数オクターブ対応
- C3 ~ C5 など、複数のオクターブを検出

## 主な機能

### リアルタイム検出
- マイク入力から音程を分析
- FFT（高速フーリエ変換）使用
- ピッチ検出アルゴリズム

### 音符表示
- 現在の音符を大きく表示
- 周波数（Hz）を表示
- 日本語と英語の両方表示

### 履歴機能
- 検出した音符の履歴
- タイムスタンプ付き
- スクロール表示

## 対応楽器

### うまく検出できる楽器
- ピアノ/キーボード
- ギター
- フルート
- リコーダー
- ボーカル（歌声）

### 検出しにくい楽器
- ドラムなどの打楽器
- 複数音が同時に鳴る和音

## ブラウザ要件

- Web Audio API対応
- マイク使用許可が必要
- 推奨：Chrome、Firefox、Edge（最新版）

## 精度向上のコツ

1. **静かな環境**: 背景ノイズを最小限に
2. **明瞭な音**: はっきりとした音で演奏
3. **単音**: 一度に1つの音を鳴らす
4. **マイクに近づく**: 適切な距離を保つ
