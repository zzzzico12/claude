# Flag Quiz Game

世界の国旗を当てるクイズゲームです。50カ国以上の国旗絵文字を使った楽しい学習アプリです。

## 特徴

- 50カ国以上の国旗クイズ
- 4択形式
- スコア記録機能
- 絵文字での国旗表示
- シャッフル機能
- リセット機能

## 技術スタック

- React (TypeScript)
- Lucide React Icons
- 国旗絵文字（Unicode）

## 使い方

1. [flag-quiz-game.tsx](flag-quiz-game.tsx)を適切なReact環境にコピー
2. 表示された国旗から国名を選択
3. 正解/不正解が即座に表示
4. スコアを確認しながらクイズを続ける
5. シャッフルで問題をランダム化

## 主な機能

### クイズモード
- 国旗が表示される
- 4つの選択肢から選ぶ
- 正解/不正解を即座にフィードバック

### スコアシステム
- 正解数をカウント
- 総問題数を記録
- 正答率を計算

### ゲーム管理
- シャッフル: 問題順をランダム化
- リセット: スコアを0に戻す
- 次の問題へ自動遷移

## UI/UX
