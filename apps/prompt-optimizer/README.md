# Prompt Optimizer

Claude 4向けにプロンプトを最適化するツールです。より効果的なプロンプトを自動生成し、AIからより良い回答を得られます。

## 特徴

- プロンプトの自動最適化
- Claude 4専用設計
- 構造化されたプロンプト生成
- ベストプラクティス適用
- Before/After比較
- ガラスモーフィズムUI

## 技術スタック

- HTML5
- Claude AI API
- JavaScript
- CSS3（ガラスエフェクト、グラデーション）

## 使い方

1. [prompt-optimizer.html](prompt-optimizer.html)をブラウザで開く
2. 元のプロンプトを入力
3. 「最適化する」ボタンをクリック
4. 最適化されたプロンプトが表示される
5. コピーボタンで簡単にコピー

## 最適化のポイント

### 構造化
- 明確な役割定義
- タスクの明確化
- 期待する出力形式の指定

### 具体性
- 曖昧な表現を避ける
- 具体的な例を含める
- 制約条件の明示

### コンテキスト
- 必要な背景情報
- 対象者の設定
- 目的の明確化

### フォーマット
- 箇条書きや番号付きリスト
- セクション分け
- 優先順位の明示

## 最適化の例

### Before（元のプロンプト）
```
料理のレシピを教えて
```

### After（最適化後）
```
# 役割
あなたは経験豊富なプロの料理人です。

# タスク
初心者向けに、以下の条件でレシピを作成してください：

## 条件
- 調理時間: 30分以内
- 材料: スーパーで手に入るもの
- 難易度: 初心者でも作れる
- 人数: 2人分

## 出力形式
1. 料理名
2. 材料リスト（分量付き）
3. 手順（番号付き）
4. 調理のコツ
```

## 主な機能

### プロンプト分析
- 現在のプロンプトの弱点を特定
- 改善ポイントを提案

### 自動最適化
- Claude 4のベストプラクティス適用
- 構造化と明確化
- コンテキストの追加

### コピー機能
- ワンクリックでコピー
- クリップボードに保存

## API設定

Claude API キーが必要です。

```javascript
headers: {
  "Content-Type": "application/json",
  "x-api-key": "YOUR_API_KEY",
  "anthropic-version": "2023-06-01"
}
```

## ベストプラクティス

### 1. 役割を明確に
「あなたは〇〇の専門家です」

### 2. タスクを具体的に
「以下の条件で〇〇してください」

### 3. 出力形式を指定
「JSON形式で」「箇条書きで」

### 4. 例を含める
「例：〇〇のような」

### 5. 制約を明示
「〇〇文字以内で」「〇〇を使わずに」
