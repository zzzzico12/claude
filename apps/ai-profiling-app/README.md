# AI Profiling App

写真をAIで解析し、創造的なプロファイリングを行うアプリケーションです。Claude APIを使用して画像から性格や趣味、コミュニケーションスタイルなどを推測します。

## 特徴

- 画像のAI解析によるプロファイリング
- 性格の特徴分析
- ライフスタイル・趣味の推測
- コミュニケーションスタイルの分析
- 美しいカードUIデザイン
- カメラ撮影・ファイルアップロード対応

## 技術スタック

- React (JSX)
- Claude AI API (画像認識・分析)
- Lucide React Icons
- Base64画像エンコーディング

## 使い方

1. [ai-profiling-app.jsx](ai-profiling-app.jsx)を適切なReact環境にコピー
2. 画像をアップロードまたはカメラで撮影
3. 「プロファイリング開始」ボタンをクリック
4. AIが画像を解析してプロファイル情報を表示

## 分析される情報

### Visual Impression（第一印象）
- 写真から受ける全体的な印象

### Personality（性格）
- 性格の特徴（4つの主要な特性）
- 性格の総合的な分析

### Lifestyle（ライフスタイル）
- 趣味・関心事（3つの項目）
- ライフスタイルの推測

### Communication（コミュニケーション）
- コミュニケーションスタイル
- 対人関係の特徴

## API設定

Claude API キーが必要です。使用する際は、適切にAPI認証を設定してください。

```javascript
headers: {
  "Content-Type": "application/json",
  "x-api-key": "YOUR_API_KEY",
  "anthropic-version": "2023-06-01"
}
```
