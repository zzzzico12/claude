# 3D Tetris

クラシックなテトリスを3D空間で楽しめるパズルゲームです。美しいグラデーション背景とガラスモーフィズムUIで、新しいテトリス体験を提供します。

## 特徴

- 3Dグラフィックスによる立体的なテトロミノ
- スコアとレベルシステム
- ガラスモーフィズムデザインのUI
- スムーズなアニメーションとエフェクト
- レベルアップによる難易度上昇

## 技術スタック

- HTML5
- Canvas API / WebGL
- JavaScript
- Three.js（3Dレンダリング）
- CSS3（ガラスモーフィズム）

## 遊び方

1. [3d-tetris.html](3d-tetris.html)をブラウザで開く
2. キーボードでテトロミノを操作
3. 横一列を揃えると消える
4. ゲームオーバーになるまで高得点を目指そう

## ゲームの操作

- ←/→ キー: テトロミノを左右に移動
- ↑ キー: テトロミノを回転
- ↓ キー: テトロミノを高速落下
- スペースキー: ハードドロップ（即座に落下）

## スコアシステム

- 1列消去: 100点
- 2列同時: 300点
- 3列同時: 500点
- 4列同時（テトリス）: 800点
- レベルが上がるごとにボーナス得点
