# Wine Menu

ワインメニューの写真をAIで解析し、データベース化できるアプリケーションです。Claude APIを使用してメニュー画像から情報を自動抽出します。

## 特徴

- ワインメニュー画像のAI解析
- 自動情報抽出（名前、産地、味わい、価格など）
- ワインの種類別フィルタリング（赤・白・スパークリング）
- 詳細情報の表示
- おすすめワインのマーキング機能
- 美しいカードUIデザイン

## 技術スタック

- React (JSX)
- Claude AI API (画像認識・テキスト抽出)
- Lucide React Icons
- Base64画像エンコーディング

## 使い方

1. [wine-menu.jsx](wine-menu.jsx)を適切なReact環境にコピー
2. ワインメニューの写真をアップロード
3. AIが自動的にメニューを解析
4. 抽出されたワイン情報を閲覧・管理

## 抽出される情報

各ワインについて以下の情報が自動抽出されます：

- **name**: ワインの日本語名
- **nameEn**: ワインの英語名
- **origin**: 産地（国と地域）
- **color**: ワインの色（赤・白・スパークリング）
- **taste**: 味わい（辛口・甘口など）
- **body**: ボディ（フルボディ・ミディアム・ライトなど）
- **description**: ワインの特徴や説明
- **price**: 価格

## 主な機能

### 画像アップロード
- カメラ撮影またはファイル選択
- ワインメニューの写真をアップロード

### AI解析
- Claude APIによる高精度な画像認識
- 複数のワイン情報を一度に抽出

### フィルタリング
- 全て表示
- 赤ワインのみ
- 白ワインのみ
- スパークリングのみ

### 詳細表示
- ワインカードをクリックで詳細情報表示
- 産地、味わい、ボディなどの詳細

## API設定

Claude API キーが必要です。使用する際は、適切にAPI認証を設定してください。

```javascript
headers: {
  "Content-Type": "application/json",
  "x-api-key": "YOUR_API_KEY",
  "anthropic-version": "2023-06-01"
}
```
