# Character Prediction Game

AI搭載の文字予測ゲームです。ユーザーが入力した文字から次に来る文字をAIが予測し、ゲーム感覚で楽しめます。

## 特徴

- リアルタイム文字予測システム
- AIによる学習機能
- スコアとストリーク（連続正解）システム
- グラデーションデザインのモダンUI
- 予測確率の表示

## 技術スタック

- HTML5
- JavaScript
- CSS3（グラデーション、アニメーション）
- AI/機械学習アルゴリズム

## 使い方

1. [claude-ai-character-prediction.html](claude-ai-character-prediction.html)をブラウザで開く
2. 入力フィールドに文字を入力
3. AIが次の文字を予測
4. 予測が当たるとスコアがアップ
5. 連続で当てるとストリークボーナス

## ゲームの楽しみ方

- AIの予測を見ながら文章を入力
- 予測確率が高い文字を選んで高スコアを狙う
- 連続正解でストリークを伸ばす
- 様々な文章パターンでAIの予測精度を試す

## スコアシステム

- 基本点: 予測が当たると10点
- ストリークボーナス: 連続正解数に応じて追加得点
- 予測確率が高いほど高評価
