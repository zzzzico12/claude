# Task Roulette

やることリストからランダムに1つを選んでくれるルーレットアプリです。何をするか迷った時に、運に任せて決められます！

## 特徴

- やることリストの管理
- ルーレットアニメーション
- ランダム選択機能
- タスクの追加・削除
- カラフルなUI
- ローカルストレージでデータ保存

## 技術スタック

- HTML5
- JavaScript
- CSS3（アニメーション、グラデーション）
- ローカルストレージAPI

## 使い方

1. [task-roulette.html](task-roulette.html)をブラウザで開く
2. やることを入力して「追加」ボタンをクリック
3. 複数のタスクを追加
4. 「ルーレットスタート！」ボタンをクリック
5. 選ばれたタスクを実行

## 主な機能

### タスク管理
- タスクの追加
- タスクの削除
- リスト表示
- 自動保存（ローカルストレージ）

### ルーレット機能
- ランダム選択アルゴリズム
- アニメーション効果
- 選択結果の強調表示
- 効果音（視覚的フィードバック）

### データ永続化
- ブラウザを閉じても保持
- 自動ロード
- 自動セーブ

### UI/UX
- グラデーション背景
- カラフルなタスクカード
- スムーズなアニメーション
- レスポンシブデザイン

## 操作方法

### タスク追加
1. 入力欄にタスクを入力
2. 「追加」ボタンまたはEnterキーで追加

### タスク削除
- 各タスクの「削除」ボタンをクリック

### ルーレット実行
- 「ルーレットスタート！」ボタンをクリック
- 2つ以上のタスクが必要

## アニメーション

ルーレット実行時：
1. タスクが順番にハイライト
2. スピードが徐々に減速
3. 最終的に1つに決定
4. 拡大表示で強調
