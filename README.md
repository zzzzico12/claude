# Claude Apps Collection

Claudeを使って作成したWebアプリケーションのコレクションです。

## 📁 プロジェクト構成

```
claude/
├── README.md
├── .gitignore
└── apps/
    ├── wine-menu/
    │   ├── README.md
    │   └── wine-menu.jsx
    └── movie-diary/
        ├── README.md
        └── movie-diary.tsx
```

## 📱 アプリ一覧

### 1. [Wine Menu](apps/wine-menu/)
- **説明**: ワインメニューを表示・管理するアプリケーション
- **技術**: React (JSX)
- **ファイル**: [wine-menu.jsx](apps/wine-menu/wine-menu.jsx)

### 2. [Movie Diary](apps/movie-diary/)
- **説明**: 映画の視聴記録を日記形式で管理するアプリケーション
- **技術**: React (TypeScript)
- **ファイル**: [movie-diary.tsx](apps/movie-diary/movie-diary.tsx)
